# Videojuegos-2MG_Ilu

Título del juego: MUCHICO
Justificación del título: "MUCHICO" es una mezcla de las palabras "Muy" (Dicho mu') y "Chico", lo que es el personaje principal.

El personaje principal, Muchico, vió un anuncio de "BIG, la bebida de los grandes". Estaban anunciando una competencia; el que llegue al final de su curso de parkour se llevaría una de estas bebidas completamente gratis. Debido al peligro de este curso, nadie tomó la oferta, pero Muchico iba a ser lo que sea para obtenerla.

Movimiento: flechas del teclado (izquierda para mover a la izquierda, derecha para mover a la derecha, arriba para saltar).
Requisitos técnicos:
  - Visual C++ Redistributable
  - Windows 11
  - Al menos 30 MB de espacio disponibles
